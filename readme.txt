Udacity Movie Website:

This program is made in Python 3, it generates an html file containing the title cover art and a trailer of 3 movies.

To run this run entertainment_center.py in python 3

You can also find the source code to this project on github at https://github.com/Ytry/Udacity-Movie-Website
